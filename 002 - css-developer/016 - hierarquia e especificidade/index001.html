Hierarquia do Efeito Cascata

Estabelece uma prioridade para aplicação de uma regra de estilo, caso sse tenha mais de uma regra para um mesmo setor CSS

Se houver um conflito, o navegador irá:

  1. Analisar todas as regras;
  2. Ver o nível de importância na origem do elemento;
  3. Comparar a especificidade das declarações com mesmo nível de importância (entra o cálculo de especificidade);
  4. En cadi de enoate di gray de especificidade, irá olhar a ordem de declaração colocada dentro dos arquivos de código. A regra CSS que for declarada por último, ira ser selecionada.
