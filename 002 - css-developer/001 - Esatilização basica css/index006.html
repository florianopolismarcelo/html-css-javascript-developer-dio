Agrupamentos de Seletores

Podemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos separar os seletores utilizando uma virgula (,).

Assim, não precisamos ficar copiando e colocando os mesmos códigos para todos, nó podemos fazer um só bloco de CSS e reutiliar as mesmas regras!
