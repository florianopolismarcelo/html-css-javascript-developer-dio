Pseuso-elementos

Usando para adicionar extras na página, sem a necessidade de declará-lo no HTML, ou para selecionar uma parte de algum elemento.

indentificado po ::palavra-chave

Pseuso-elementos - ::first-letter

Selecione a primeira letra da primeira linha

Propriedades disponíveis:

  • color
  • background
  • margin
  • padding
  • border
  • font
  • propriedades de texto
